Detailed Description

Content provider for the Tekdaqc. Provides a persistent data store for analog data and calibration data. By default, this provider uses DefaultTekdaqcDatabaseHelper as its internal helper. If desired, user code can override this by calling setDatabaseHelperClass(Class) PRIOR to the content provider being loaded.

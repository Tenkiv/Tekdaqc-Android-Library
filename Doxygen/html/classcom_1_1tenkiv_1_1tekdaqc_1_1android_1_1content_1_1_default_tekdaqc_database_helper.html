Detailed Description

Default implementation of the ATekdaqcDatabaseHelper. This is simply a thin wrapper around the abstract class, allowing the use of the default parameters. The database will be created at the location specified by ATekdaqcDatabaseHelper#DEFAULT_FILE_DIR.
